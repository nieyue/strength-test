一、账号注册

1. 用户在使用本服务前需要注册一个 账号。该账号应当使用手机号码绑定注册，用户须使用尚未与 账号绑定的手机号码，以及未被嘉行公司根据本协议封禁的手机号码注册 账号。嘉行公司可以根据用户需求或产品需要对账号注册和绑定的方式进行变更，而无须事先通知用户。

2. 系列应用产品，所有权属于嘉行公司，因用户注册时应当授权嘉行公司公开及使用其个人信息方可成功注册，故用户完成注册即表明用户同意嘉行公司提取、公开及使用用户的信息。

3. 鉴于 账号的绑定注册方式，您在注册时即视为已同意嘉行公司在注册时将允许您的手机号码及手机设备识别码等信息用于注册。

4. 在用户注册及使用本服务时，嘉行公司需要搜集能识别用户身份的个人信息以便嘉行公司可以在必要时联系用户，或为用户提供更好的使用体验，嘉行公司搜集的信息包括但不限于用户姓名、地址；嘉行公司同意对这些信息的使用将受限于第三条用户个人隐私信息保护的约束。

5. 若用户注册后连续（二个月）不登录账号进行活动，即视为僵尸账号，嘉行公司有权单方将用户账户中的余额清零，账号进行初始化。

二、用户信息保护

1. 如果嘉行公司发现或者收到他人举报或投诉用户违反本协议约定的或用户存在违反法律法规及政策等情形，嘉行公司有权不经通知随时对相关内容，包括但不限于用户资料、发帖记录进行审查、删除，并视情节轻重对违规账号处以包括但不限于警告、账号封禁、设备封禁、功能封禁的处罚，并通知用户处理结果。

3. 用户理解并同意，嘉行公司有权依合理判断对违反有关法律法规或本协议的行为进行处理，对违法违规的任何用户采取适当的法律行动，并依据法律法规保存有关信息向有关部门报告等，用户应承担由此而产生的一切法律责任。

4. 用户理解并同意，因用户违反本协议约定或法律法规及政策规定，导致或产生的任何第三方主张的任何索赔、要求或损失，包括合理的律师费，用户应当赔偿嘉行公司与合作公司、关联公司。

（3）损害国家荣誉和利益的；

（6）散布谣言，扰乱社会秩序，破坏社会稳定的；
